Detailed Description

GeometricEditor is a Java graphical application for drawing and editing geometric shapes such as circles, rectangles, and polygons. It supports saving/loading, changing color, moving, scaling, and rotating shapes.

Author: Adam Kulwicki The main class of the GeometricEditor application, which allows drawing and editing geometric shapes.
